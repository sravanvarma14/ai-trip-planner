# ✈️ VoyagerAI — Intelligent Trip Planner

An AI-powered personalized trip planner that dynamically creates end-to-end itineraries tailored to individual budgets, interests, and real-time conditions with seamless booking capabilities.

![VoyagerAI](https://images.unsplash.com/photo-1476514525535-07fb3b4ae5f1?w=1200&q=80)

## 🌟 Features

- **🤖 AI-Powered Itineraries** — Smart day-by-day trip plans personalized to your preferences
- **🌍 40+ Destinations** — Curated destinations across Europe, Asia, Americas, Africa & Oceania
- **💰 Budget Optimization** — Plans tailored to your budget from $500 to $15,000+
- **📊 Real-Time Dashboard** — Weather, crowd levels, local events & safety scores
- **🎫 Seamless Booking** — Book hotels, activities & dining directly within your itinerary
- **🎲 Surprise Me!** — Let AI pick a random destination for you
- **❤️ Save Trips** — Save and revisit your favorite itineraries
- **📱 Fully Responsive** — Works beautifully on desktop, tablet & mobile

## 🚀 Getting Started

Simply open `index.html` in your browser — no build tools or server required!

```bash
# Clone the repository
git clone https://github.com/sravanvarma27/AI-Trip-Planner.git

# Open in browser
open index.html
```

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, glassmorphism, gradients, animations

- **Google Fonts** — Inter & Outfit
- **Font Awesome 6** — Icons
- **Unsplash** — Beautiful travel photography

## 📋 How It Works

1. **Choose a Destination** — Search, pick from 40+ quick picks, or hit "Surprise Me!"
2. **Set Dates & Budget** — Use the interactive slider and date pickers
3. **Select Interests** — Pick from 15 interest categories
4. **Fine-tune Preferences** — Travel pace, accommodation style, dietary needs
5. **Generate Itinerary** — Watch AI craft your perfect trip
6. **Book & Go!** — Reserve everything directly in the app
